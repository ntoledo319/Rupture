Lambda Node.js 20 Phase 1

AWS Lambda — end-of-life deadline

2026-04-30

This deadline passed on 2026-04-30. Affected resources are now in the post-deadline window — clean up before the next enforcement phase.

What is being deprecated?

Lambda functions using Node.js 20 runtime will fail to update. Phase 1: CreateFunction and UpdateFunctionConfiguration blocked.

Breaking changes

The following changes take effect at or before the 2026-04-30 deadline. [source]

  • CreateFunction and UpdateFunctionConfiguration API calls fail
  • aws-sdk v2 not available (use v3)
  • OpenSSL 3 hash changes
  • URL vs require() for imports
