IMDSv1 Enforcement

EC2 — end-of-life deadline

2025-12-31

This deadline passed on 2025-12-31. Affected resources are now in the post-deadline window — clean up before the next enforcement phase.

What is being deprecated?

IMDSv1 access will be blocked by default on new instance launches.

Deadline2025-12-31 (-152 days from build date 2026-05-31)
ServiceEC2
Affected resourcesinstance|launch-template
Severityhigh
Primary sourcehttps://docs.aws.amazon.com/AWSEC2/latest/UserGuide/configuring-instance-metadata-service.html

Breaking changes

The following changes take effect at or before the 2025-12-31 deadline. [source]

  • IMDSv1 requests fail
  • Must use IMDSv2 with session tokens
  • SSM agent must be updated
