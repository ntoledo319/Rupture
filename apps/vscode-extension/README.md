# Rupture - AWS Deprecation Scanner

Rupture scans CloudFormation, SAM, Terraform, JavaScript, TypeScript, Python, and JSON files for AWS runtime and platform deprecation risk.

## Commands

- `Rupture: Scan Workspace for Deprecations`
- `Rupture: Show Deprecation Report`
- `Rupture: Get Full Audit Report`

## Settings

- `rupture.enabledKits`
- `rupture.severityThreshold`
- `rupture.autoScan`

The extension runs locally in VS Code and links to the hosted audit flow for deeper reporting.
